# Finance Skills

A collection of Claude Code skills for financial analysis and trading.

## Structure

Each skill lives in its own directory under `skills/`:

```
skills/
  options-payoff/
    SKILL.md          # Skill definition (frontmatter + instructions)
    references/       # Supporting reference files
```

## Adding a new skill

1. Create a directory under `skills/` with the skill name
2. Add a `SKILL.md` with YAML frontmatter (`name`, `description`) and instructions
3. Add any reference files under `references/`
